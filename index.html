Planejador de viagens

Vacation Planner

Um aplicativo web para gerenciar suas viagens utilizando React, Typescript, Shadcn/ui, Tailwind CSS, React Hook Form, Zod e date-fns. Esse aplicativo foi criado a partir do problema que eu estava tendo de gerar as viagens que meus familiares queriam fazer e não sabiam como organizar direito. Dessa forma criei a plataforma sendo totalmente responsiva para que pudessem gerenciar tanto no telefone quanto no computador.
